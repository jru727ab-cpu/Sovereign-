# CIVILTAS Progression System

## XP Sources

| Action | XP Reward |
|--------|-----------|
| Daily check-in | 25 |
| Collect offline earnings | 20/hr (capped 8hr) |
| Complete daily task | 50–150 |
| Upgrade a building | 50 |
| Unlock a building | 100 |
| Decode a relic | 200 |
| Catastrophe survival | 500 |

## Level Milestones

| Level | Unlock |
|-------|--------|
| 1 | Mine (default) |
| 3 | Farm |
| 5 | First Inner Circle message; Library |
| 8 | Oil Derrick |
| 10 | Vault |
| 15 | Watchtower + catastrophe forecast hint |
| 20 | Bunker upgrades |
| 25 | Shrine (secret society interaction) |
| 30 | Catastrophe cycle full reveal |
| 40 | Market (trade with other survivors — future) |
| 50 | True ending / prestige reset |

## Secret Society (Inner Circle) Ranks

| Rank | Name | Requirement |
|------|------|-------------|
| 0 | Outsider | Default |
| 1 | Initiate | Level 5 + decode 1 Relic |
| 2 | Seeker | Level 15 + decode 5 Relics |
| 3 | Adept | Level 25 + Gnosis > 100 |
| 4 | Illuminated | Survive 3 catastrophes |
| 5 | Architect | Level 50 + all buildings max level |
